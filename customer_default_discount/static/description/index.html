Few Words About The Customer Based Discount

Customer Based Discount or Customer Default Discount module will help you to set default and specific discount for customers and that discount will apply on sale orders and invoices. You can also set a different discount then the default discount for a order or invoice also. It will be more handy to apply discounts for your customers. Default sales discount

Work Flow: How The Customer Based Discount Works
Set default discount for specific customer
Default Discount in Sales Order
Default Discount in Invoice
